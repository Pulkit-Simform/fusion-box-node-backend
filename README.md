## Description

FusionWorks unifies HR & project management. Manage resources, track allocations, and get AI-powered answers to HR questions. Eliminate repetitive tasks and boost efficiency and a streamlined platform.

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ yarn install
```

## Running the app

```bash

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e
```
